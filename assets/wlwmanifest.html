Razlog za zaščito Windows Live Writer (wlwmanifest)

Zaradi večjega obsega napadov na Wordpress funkcionalnost Windows Live Writer (wlwmanifest) je v veljavi zaščita za dostop do teha mehanizma.


Kako ukrepati?

Za posamezno WordPress stran je dostop do Windows Live Writer (wlwmanifest) še vedno možno omogočiti z ustrezno direktivo v ".htaccess" datoteki spletne strani.

Primer take direktive:

# BEGIN Allow Windows Live Writer (wlwmanifest)
<FilesMatch "(wlwmanifest)\.xml$">
	Require all granted
</FilesMatch>
# END Allow Windows Live Writer (wlwmanifest)
